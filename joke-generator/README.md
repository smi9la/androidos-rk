# Random Joke Generator

A simple web application that fetches and displays random jokes from an external API.

## Features

- 🎭 Fetches jokes from a public API
- 🎨 Clean, modern UI
- 🔄 One-click joke generation
- 📱 Responsive design
- ⚡ Fast loading
- 🎯 Multiple joke types support

## APIs Supported

1. **JokeAPI** (https://jokeapi.dev)
   - Multiple joke types
   - Filtering options
   - Reliable & fast

2. **Official Joke API** (https://official-joke-api.appspot.com)
   - Simple jokes
   - Random selection
   - Easy integration

## Quick Start

```bash
# Clone repository
git clone https://github.com/smi9la/joke-generator.git
cd joke-generator

# Open in browser
open index.html
# or
firefox index.html
```

## Usage

1. Open `index.html` in your web browser
2. Click "Get Joke" button
3. Wait for joke to load
4. Enjoy! 😂
