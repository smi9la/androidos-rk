3. Follow the installation wizard
4. Launch from Start Menu or Desktop

### Using Portable Version
1. Download `AndroidOS-Clock-1.0.0.exe`
2. Run directly (no installation needed)
3. Optional: Create shortcut to desktop

### Requirements
- Windows 7 or higher
- 200 MB free disk space
- .NET Framework (usually pre-installed)

## 📱 Android Installation

1. Download `androidos-clock-1.0.0.apk`
2. Transfer to your Android device (USB or Email)
3. Open file manager and locate the APK
4. Tap to install (may need to enable "Unknown sources" in Settings → Security)
5. Launch from App Drawer

### Requirements
- Android 5.0 (API 21) or higher
- 50 MB free storage
